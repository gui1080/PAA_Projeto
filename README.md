# Projeto de Disciplina PAA

## Como rodar

É necessário ter o scrapy instalado e ser capaz de executá-lo. Numa máquina linux roda

```
make install
```

Em seguida para rodar o web-crawler usa-se

```
scrapy runspider scrapy/scraper.py
```
